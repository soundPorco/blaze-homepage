SKILLIMPROVE LABO

県内初のスクール専用施設

徳島県初となるバスケットボールスクール専用施設として誕生した SKILIMPROVE LABO は、 BASKETBALL SCHOOL BLAZEの新たな拠点です。子どもたちが集中して練習し、のびのびと成長できる環境を整えています。

OUTLINE

施設概要

  • 徳島市北沖洲2丁目164
  • コート面積200㎡
  • 高さ調整可能なバスケットリング3基
  • シューティング専用マシン
  • トレーニング用具完備
  • 保護者見学スペース・冷風機
  • 仮設水洗トイレ

CLASS INTRODUCTION

成長を加速させる、クラスラインナップ

「もっと上手くなりたい」「バスケの楽しさを知りたい」「基礎をしっかり学びたい」
BLAZE BASKETBALL SCHOOL では、そんなすべての想いに応えるクラスをご用意しています。

クラスは、目的やスキルに応じて段階的に設計されており、初心者が安心してスタートできるベーシッククラスから、実戦力を磨くアドバンスドクラスまで幅広く対応しています。 指導は丁寧にサポートするため、初めてバスケットボールに触れるお子様も、ステップアップを目指す経験者も、ここでなら着実な成長を実感できるはずです。

受講者の経験、目的に応じて複数のコースをご用意しています。 初心者から上級者まで、安心して学べる充実したカリキュラムで、確かな成長をサポートします。 それぞれのニーズに合った最適なコースをぜひご覧ください。 さあ、自分にぴったりのクラスを見つけて、一緒に成長への一歩を踏み出しましょう！

CLASS TYPES

各クラスの紹介

BASIC CLASS

バスケの基礎を楽しく学ぶ入門クラス

身体の使い方、ボールハンドリング、正しいシュートフォームなど、 バスケットボールの基本技術を楽しく学べる入門クラスです。
ファンダメンタル（基礎）の習得を通じて、スキルの土台をしっかり築いていきます。

こんな人におすすめ

  • ● バスケ初心者（U12〜U15）
  • ● 基礎から丁寧に学びたい方
  • ● チーム練習だけでは補えないスキルを身につけたい方

バスケットボールの楽しさは、確かな基礎から始まります！

このBASIC CLASSは、バスケットボールを始めたばかりのお子様や、もう一度基礎からしっかりと学び直したいお子様に最適なクラスです。 スキルの土台となるファンダメンタル、つまり「身体操作」「ボールの扱い」「シュート」といったバスケットボールの根幹を、時間をかけて丁寧に構築していきます 。
一人ひとりのレベルに合わせて指導を行うことで、正しいフォームと動きを習得し、バスケットボールの楽しさを深く実感できるでしょう。このクラスで身につけた確かな基礎は、お子様のバスケットボール人生を豊かにするかけがえのない財産となります。

ADVANCED CLASS

判断力と実戦的スキルを鍛える競技特化型クラス

実戦を想定した状況判断やプレー選択を含めた、より高度なスキルの習得を目指すクラスです。
実践的なトレーニングを通じて、競技力の向上に特化した内容を提供します。

こんな人におすすめ

  • ● U15 または U12 の上級者
  • ● 実戦で通用する判断力と応用スキルを高めたい選手
  • ● より競技性の高い環境でステップアップしたい方

次のレベルへ！実践で活きるスキルを習得し、試合で輝こう！

ADVANCED CLASSは、バスケットボールの経験があり、さらに競技力を向上させたいと願うお子様のための、より実践的で競技に特化したクラスです 。
このクラスでは、単なる技術の習得に留まらず、「判断を伴う実戦的なスキル」の養成に力を入れています 。試合中に刻々と変化する状況を正確に判断し、適切なプレーを選択する能力、そしてそれを実行する高度なスキルを磨きます。より複雑な戦術やチームプレーにも挑戦し、個々の能力をチームの中で最大限に発揮できるよう指導します。
将来、大会での活躍を目指すお子様にとって、このクラスはまさに飛躍の場となるでしょう。

PRACTIVE TIME

練習時間について

BLAZE BASKETBALL SCHOOL では、1回の練習を90分間（1時間半）たっぷり確保しています。 基礎から応用までしっかり取り組める時間設定で、初心者から経験者まで満足いただける内容です。 平日の中から週5日（月曜日〜日曜日のうち5日間）、 ご都合の良いスケジュールに合わせて、以下の2つの時間帯からお選びいただけます。

  • ① 18:15〜19:45
  • ② 20:00〜21:30

練習が始まる15分前から専用コートを開放しています。 ウォームアップをしたり、友だちとフリーシュートを楽しんだり、自分のペースで準備ができるのも魅力です。
「学校や習い事の後でも通える時間帯」 「毎週同じ曜日じゃなくても大丈夫」 そんな柔軟な運営を心がけているので、無理なく、長く続けやすい環境を整えています。
